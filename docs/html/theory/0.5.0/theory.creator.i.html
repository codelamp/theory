theory.creator.i.processItems(obj) (static)

Creators can call a number of other Theory functionalities into play just by defining subobject in particular ways. This function is what reacts to those subobjects. Currently supported are:

  • wrappers — See theory.wrapper
  • methods — See theory.method
  • overloads — See theory.overload
